# Laundry Service Website

Hi,

This project is a simple HTML and CSS webpage created as part of my practice.
The goal was to design a clean and modern laundry service homepage with a responsive layout and a CSS-based hamburger menu.

## What I Made

I created a laundry service website that includes:
• Navigation bar (Logo, Menu, User)
• Responsive hamburger menu (pure CSS)
• Hero section with heading and description
• Laundry image
• Call-to-action button

## How to Run the Project

1. Download or open the project folder.
2. Make sure all files are in the same folder:
   * index.html
   * style.css
   * machine.jpg
3. Double-click on **index.html**.
4. The page will open in your browser.

## Files in This Folder

• index.html → Main webpage
• style.css → Styling file
• machine.jpg → Image used
• README.md → Project details
